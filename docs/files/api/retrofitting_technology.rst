RetrofittingTechnology
======================

Overview
--------

``RetrofittingTechnology`` models retrofit options for existing technologies.

Use Cases
---------

- Represent upgrade pathways for existing infrastructure.
- Define retrofit-specific attribute behavior and constraints.
